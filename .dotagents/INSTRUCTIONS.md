## Project

Cyrus is a cross-platform app for controlling AI coding agents (Claude Code, Codex, and others) that run across a user's own devices — desktop, mobile, web, laptops, and servers.

It is single-user and personal: every device a user owns joins that user's one "room." Thread metadata is shared across all devices through a minimal server, while agent execution stays local to the device that owns it. The guiding principle is **shared metadata, owned execution** — letting the user start, observe, and continue the same agent conversation from any of their devices in real time.
